The Components are available in three forms:

  • bc.tar.gz, normally for UNIX systems:
    tar zxvf bc.tar.gz or
    zcat bc.tar.gz | tar xvf -
  • bc.zip, normally for Windows 95 (long file names). Use WinZip or equivalent (has CR/LF line terminations)
  • bc.src, uncompressed concatenated source, for DOS/Windows 3.
    Use the GNAT utility gnatchop to extract the sources and crunch the file names to 8.3 form.
    GNAT can be obtained from ftp://cs.nyu.edu/pub/gnat and mirrors.
