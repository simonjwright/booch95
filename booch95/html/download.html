Getting Releases

The Components are available in three forms:

  • bc-yyyymmdd.tgz (.tar.gz), normally for UNIX systems:
    tar zxvf bc-19980924.tgz or
    zcat bc-19980924.tgz | tar xvf - or
    gunzip < bc-19980924.tgz | tar xvf -
  • bc-yyyymmdd.zip, normally for Windows 95 (long file names). Use WinZip or equivalent (has CR/LF line terminations)
  • bc-yyyymmdd.src, uncompressed concatenated source, for DOS/Windows 3.
    Use the GNAT utility gnatchop to extract the sources and crunch the file names to 8.3 form.
    GNAT can be obtained from ftp://cs.nyu.edu/pub/gnat and mirrors.
