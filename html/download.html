Download

Releases of the Components are available in two forms:

  • bc-yyyymmdd.tgz (.tar.gz), normally for UNIX systems:
    tar zxvf bc-19980924.tgz or
    zcat bc-19980924.tgz | tar xvf - or
    gunzip < bc-19980924.tgz | tar xvf -
  • bc-yyyymmdd.zip, normally for Windows 95 (long file names). Use WinZip or equivalent. Has CR/LF line terminations.

The HMTL documentation is included in the distribution.

The Sourceforge repository can be viewed, or checked out using for example

   svn checkout http://svn.code.sf.net/p/booch95/code/trunk booch95

GNAT can be obtained from AdaCore or as part of GCC from the Free Software Foundation.
